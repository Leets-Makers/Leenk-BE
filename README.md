## 📱 서비스 소개 
#### "우리가 연결되는 순간, Leenk로 시작해봐!"
동아리 활동을 하다 보면 이런 고민, 한 번쯤 해본 적 있지 않으신가요? 
<br/>

💭 모임은 있지만 더 자주 만나고 싶을 때 <br/>
🥺 이미 친해진 무리에 끼어들기 어려울 때 <br/>
🌱 새로운 사람들과 친해지고 싶은데 계기가 없을 때 
<br/>

Leenk는 이런 순간을 자연스럽게 연결해주는 서비스예요.
<br/>

우리들만의 SNS, **Leenk**와 함께 더 쉽게 즐겁게 동아리 생활의 새로운 연결을 시작해보세요🙌🏻
